# LinkedIn Personal Brand OS

A free Claude Code skill that turns Claude into a personal LinkedIn branding coach.

Based on the Lara Acosta 4-3-2-1 framework — the system used to grow from 0 to 300K+ followers and build multiple 7-figure businesses through LinkedIn content alone.

## What it does

When installed, Claude becomes your personal LinkedIn coach. It knows:

- The full 4-3-2-1 content system (4 posts/week, 3 pillars, 2 audiences, 1 lead magnet)
- Hook formulas that stop the scroll
- LinkedIn profile audit + rewrite (headline, banner, about, featured)
- The SLAY edu-telling framework
- IFP vs ICP distribution strategy
- Lead magnet builder
- DM-to-lead conversion playbook
- 23 reference tables covering algorithm, social proof, engagement, storytelling

## Install (3 minutes)

### Step 1 — Install Claude Code

Download from [claude.ai/code](https://claude.ai/code). Mac or Windows.

### Step 2 — Create the skill folder

**Mac (Terminal):**
```bash
mkdir -p ~/.claude/skills/linkedin-personal-brand
```

**Windows (Command Prompt):**
```cmd
mkdir %USERPROFILE%\.claude\skills\linkedin-personal-brand
```

### Step 3 — Add the skill file

Download `SKILL.md` from this repo and place it inside the folder you just created.

Final path:
- Mac: `~/.claude/skills/linkedin-personal-brand/SKILL.md`
- Windows: `C:\Users\[yourname]\.claude\skills\linkedin-personal-brand\SKILL.md`

### Step 4 — Restart Claude Code

Close and reopen the app.

### Step 5 — Test

Type into Claude Code:

```
I want to build my LinkedIn personal brand
```

Claude will respond as a personal brand coach and start asking discovery questions.

## How to use it

The skill activates automatically when you talk about LinkedIn, content, personal brand, positioning, hooks, ICP, or content pillars.

### Best starting prompts

**Run a full strategy session:**
```
I'm a [your role/business]. Run me through your full positioning and strategy session. Ask one question at a time. Don't explain the framework — just ask the questions.
```

**Audit your profile:**
```
Audit my LinkedIn profile. For each section give me: STATUS (Pass/Fix/Missing), the issue, and the exact rewrite ready to paste.

Headline: [paste]
About: [paste]
URL: [your LinkedIn URL]
```

**Write a post from a story:**
```
Here's a story from my business: [tell the story]. Find the 3 strongest angles, write 4 hook variations each, label IFP or ICP, pick the strongest, then write the full post using the SLAY framework.
```

**Plan your week:**
```
Plan my LinkedIn content for the next 7 days using the 4-3-2-1 system. 4 posts (Tue/Wed/Thu/Sat). For each: post type, pillar, audience, topic, 2 hooks, key points, CTA.
```
